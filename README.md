# 🚩 youth_camp_low-code 

起于青训营，止于想象。在青训营结识伙伴，一起完成有意思的事情，虽此时功夫不深，然路漫漫其修远昔，吾将上下而求索。

## 📝项目介绍
### 🔑开发环境配置

借助vue-cli 脚手架进行开发



### 📌项目演示地址

[Demo](https://leoyongyuan.github.io/YouthCamp_Low-Code/dist/index.html#/)


### ▶如何运行

####  安装

```
npm install
```

#### 运行

```
npm run dev
```

#### 打包

```
npm run build
```
